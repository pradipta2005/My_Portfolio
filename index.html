Plotiva | No-Code Interactive Data Visualization & ML App

Overview:
Plotiva is a Streamlit-powered web application that empowers users to perform data exploration, visualization, and machine learning—all without writing a single line of code.

Key Features:

  • Data Ingestion & Management: Upload CSV files or create datasets within the app.
  • Dynamic Charts: Instantly generate a variety of interactive visualizations—bar charts, line graphs, scatter plots, area charts, pie charts—leveraging native Streamlit capabilities along with Plotly, Altair, Matplotlib, and Vega‑Lite.
  • Interactive Plotly Integration: Offers hover tooltips, selection support, zooming/panning, and theme customization for deep data interaction.
  • Layout & UI Components: Clean UI with sidebar widgets (dropdowns, sliders, file uploader), tabbed navigation, and responsive layout for intuitive usage.
  • Advanced Features: Utilizes session state and Streamlit caching to preserve user inputs and optimize performance.
  • Modeling Support: Enables no-code application of common ML workflows (e.g., linear/logistic regression, classification, clustering), includes fitting models and previewing results.
  • Dashboard Assembly: Allows users to compile multiple visualizations into cohesive, shareable dashboard layouts.
  • Deployment & Hosting: Deployed live on Streamlit Community Cloud, ready for exploration and feedback.
